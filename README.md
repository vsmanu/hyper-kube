# hyper-kube
A super charger for kubernetes.

hyper-kube is a boilerplate which coins contemporary kube addons togather in a single key-stroke. 

| Function | Tool|
----------| -- |
| Gitops   | Argocd |
| OAM      | Kubevela |
| Certificates | Certmanager |
| Networking | Calico |
| Ingres | traefik |
| Monitoring | thanos |
| Visualization | Grafana |
| Secrets | Sealed secrets |
| Service proxy | Envoy |
| Logging | Fluentd |
